## Implementation Notes

**Data Points Used**

- Title length (max 60 characters)
- Meta description length (max 160 characters)

**Formula**

CTR is estimated with a simple weighted model:

$$
\text{CTR} = 0.1 + 0.4 \times \frac{\min(\text{title\_len}, 60)}{60} + 0.5 \times \frac{\min(\text{desc\_len}, 160)}{160}
$$

The on-page preview explains each contribution to this score.
