# ROAS & Reality Check Calculator


ROAS (Return on Ad Spend) measures how much revenue is earned for every dollar spent. The formula is $\text{ROAS}=\frac{\text{Revenue}}{\text{Ad\,Spend}}$.

Use this tool to estimate profitability across different conversion rate scenarios.
